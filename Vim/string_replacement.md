# 문자열 치환하는 방법
### 명령어 기본
```bash
: 시작 행,끝 행s/찾을 패턴/바꿀 문자열/옵션
```

- 찾을 패턴에는 정규 표현식도 가능하다.

- 문자열에 `/` 을 사용하고 싶다면 앞에 `\` 을 붙이면 된다.

### 범위 지정
- 특정 행 범위 내에서 지정할 땐 숫자로 나타낸다.
    - ex) 1행부터 10행까지 : **`1, 10s/`**

- 현재 커서가 위치한 행은 **`.`** 으로 나타낸다.
    - ex) 1행부터 현재 커서 행까지 : **`1, .s/`**

- 마지막 행은 **`$`** 로 나타낸다.
    - ex) 현재 커서 행부터 마지막 행까지 : **`., $s/`**

- 문자 전체에서 바꾸고 싶을 때는 **`%`** 를 사용한다.
    - ex) **`%s/`**

### 옵션 종류
- **`g`** : 한 줄에 패턴이 여러 번 나오면 모두 변경한다. 지정하지 않으면 첫 번째 패터만 변경한다.

- **`i`** : 대소문자를 구분하지 않고 변경한다.

- **`c`** : 매칭되는 문자열마다 바꿀 것인지 사용자 사용자에게 물어보고 변경한다.
